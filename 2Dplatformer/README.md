1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Project Structure

- `src/game/` - Core game logic
- `public/assets/` - Game assets (sprites, audio, tilemaps)
- `docs/` - Documentation and guides
